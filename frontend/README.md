# Agentic Platform Frontend

Vue 3 frontend for the Agentic Platform MVP.

## Setup

```bash
npm install
```

## Development

```bash
npm run dev
```

## Build

```bash
npm run build
```

## Tutor Tool

The Tutor Tool frontend now starts with:

- Subject
- Academic level (`high_school`, `college`, `phd`)
- Main action selection: `Ask Question`, `Upload Notes`, `Practice`

The Tutor workspace renders structured AI results with explanation, steps, practice, summary for uploaded notes, and a persisted progress sidebar.

## Testing

```bash
npm run test:unit
```
